# GEO5017 - Assignment 1: Linear Regression

This repository contains the implementation for Assignment 1 of the GEO5017: Machine Learning for the Built Environment course. The project focuses on solving linear and polynomial regression problems using a custom Gradient Descent solver to track drone trajectories.

## Project Structure
- `assignment1/main.py`: The primary entry point for the assignment tasks.
- `assignment1/GEO5017-A1-LinearRegression.pdf`: The assignment instructions and problem statement.
- `pyproject.toml` / `uv.lock`: Configuration files for dependency management.
- `.python-version`: Specifies the required Python version (3.12).

## Prerequisites
This project requires **Python 3.12** or higher. It uses the following libraries:
- `numpy`: For numerical computations and matrix operations.
- `torch`: For advanced mathematical modeling.
- `scikit-learn`: For comparative analysis (if applicable).
